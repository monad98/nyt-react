## Quick Start
```bash
#clone the repo
git clone https://github.com/monad98/nyt-react.git

#change directory to repo
cd nyt-react

# install dependencies
npm install

# build
npm run build

#start
npm run serve
```

## Packages used
- Express
- MongoDB
- React
- Redux
- socket.io
